## 🚀 เริ่มต้นใช้งาน / Getting Started

### 1. เปิดเว็บไซต์

เปิดเบราว์เซอร์แล้วเข้า:

```
http://localhost/Wikanda_Hair_Salon/public/
```

| บทบาท       | ชื่อผู้ใช้ | รหัสผ่าน      |
| ----------- | ---------- | ------------- |
| ผู้ดูแลระบบ | `admin`    | `password123` |
| เจ้าของร้าน | `owner`    | `password123` |
| พนักงาน     | `staff01`  | `password123` |
| พนักงาน     | `staff02`  | `password123` |
| สมาชิก      | `member01` | `password123` |
| สมาชิก      | `member02` | `password123` |

## 👤 สำหรับสมาชิก / For Members

### จองคิว / Book an Appointment

1. คลิก **"จองคิวเลย"** จากหน้าแรก
2. เลือก **บริการ** (เช่น ตัดผม, ทำสี)
3. เลือก **ช่าง** ที่ต้องการ
4. เลือก **วันที่** และ **เวลา**
5. กด **"จองคิว"**

### ดูการจองของฉัน / My Bookings

1. คลิกชื่อผู้ใช้ที่มุมขวาบน → **"การจองของฉัน"**
2. จะเห็นรายการจองทั้งหมด พร้อมสถานะ
3. ถ้าสถานะเป็น **"รอดำเนินการ"** หรือ **"ยืนยันแล้ว"** สามารถกด **"ยกเลิกการจอง"** ได้

### แก้ไขโปรไฟล์ / Edit Profile

1. คลิกชื่อผู้ใช้ → **"โปรไฟล์ของฉัน"**
2. แก้ไขอีเมล, ชื่อ, เบอร์โทร หรือรหัสผ่าน
3. กด **"บันทึกการเปลี่ยนแปลง"**

---

## 💇 สำหรับพนักงาน / For Staff

### ดูการจองวันนี้ / Today's Bookings

1. เข้าสู่ระบบด้วยบัญชี staff
2. หน้าแดชบอร์ดจะแสดงการจองของวันนี้ทันที
3. คลิก **"การจองของฉัน"** เพื่อดูทั้งหมด

---

## 🛠️ สำหรับผู้ดูแลระบบ / For Admin

### แดชบอร์ดผู้ดูแล / Admin Dashboard

1. เข้าสู่ระบบด้วยบัญชี admin หรือ owner
2. จะเห็นสรุปภาพรวม:
   - จำนวนการจองวันนี้
   - รายได้วันนี้
   - รายการรอดำเนินการ

### จัดการการจอง / Manage Bookings

1. คลิก **"จัดการการจอง"** จากเมนู
2. เปลี่ยนสถานะการจองได้จาก dropdown:
   - **รอดำเนินการ** → ยืนยันแล้ว → กำลังให้บริการ → เสร็จสิ้น
   - หรือ **ยกเลิก** ถ้าลูกค้าไม่มา

### จัดการการชำระเงิน / Manage Payments

1. คลิก **"จัดการการชำระเงิน"**
2. ถ้าลูกค้าส่งสลิปมา:
   - กด **"อนุมัติ"** ถ้าสลิปถูกต้อง
   - กด **"ปฏิเสธ"** ถ้าสลิปไม่ถูกต้อง

### ดูรายงาน / Reports

1. คลิก **"รายงาน"**
2. ดูได้:
   - รายได้วันนี้
   - รายได้เดือนนี้ (แยกตามวัน)
   - บริการยอดนิยม

### จัดการบริการ / Manage Services

1. คลิก **"จัดการบริการ"**
2. ดูรายการบริการทั้งหมด พร้อมราคาและระยะเวลา

### จัดการช่าง / Manage Staff

1. คลิก **"จัดการช่าง"**
2. ดูข้อมูลช่างทั้งหมด:
   - ความเชี่ยวชาญ
   - วันทำงาน
   - ประสบการณ์

### จัดการผู้ใช้งาน / Manage Users

1. คลิก **"จัดการผู้ใช้งาน"**
2. ดูรายการผู้ใช้ทั้งหมด พร้อมบทบาท

## 💰 การชำระเงิน / Payments

### วิธีการชำระเงินที่รองรับ

| วิธี          | รายละเอียด               |
| ------------- | ------------------------ |
| **PromptPay** | โอนเงินแล้วแนบสลิป       |
| **โอนเงิน**   | โอนผ่านธนาคารแล้วแนบสลิป |
| **เงินสด**    | จ่ายที่หน้าร้าน          |

### ประเภทการชำระเงิน

- **มัดจำ (Deposit)** — จ่ายล่วงหน้า 100 บาท
- **เต็มจำนวน (Full)** — จ่ายครบตอนจองหรือหลังใช้บริการ

---

## 📱 การแจ้งเตือน LINE / LINE Notifications

- ระบบสามารถส่งข้อความแจ้งเตือนไป LINE ของลูกค้าได้
- ต้องเปิดใช้งานใน `config/line.php` และใส่ Channel Access Token
- ถ้ายังไม่เปิดใช้งาน ระบบจะบันทึก log แทน

---

## 🔒 ความปลอดภัย / Security

- รหัสผ่านถูก hash ด้วย bcrypt
- Session หมดอายุใน 2 ชั่วโมง
- ทุกการเปลี่ยนแปลงข้อมูลต้องผ่านการยืนยันตัวตน
- สมาชิกแก้ไขได้แค่ข้อมูลตัวเอง
- แอดมิน/เจ้าของ จัดการทุกอย่างได้

## ❓ แก้ไขปัญหาเบื้องต้น / Troubleshooting

### หน้าเว็บไม่โหลด / Page Not Found

1. ตรวจสอบว่า Apache กำลังรันอยู่ใน XAMPP
2. ตรวจสอบว่า `mod_rewrite` เปิดใช้งานแล้ว
3. ตรวจสอบว่าโปรเจกต์อยู่ใน `C:\xampp\htdocs\Wikanda_Hair_Salon\`

### เข้าสู่ระบบไม่ได้ / Can't Login

1. ตรวจสอบชื่อผู้ใช้และรหัสผ่าน
2. รหัสผ่านทดสอบทั้งหมดคือ `password123`
3. ถ้าลืมรหัสผ่าน ติดต่อแอดมิน

### จองคิวไม่ได้ / Can't Book

1. ตรวจสอบว่าเลือกบริการและช่างแล้ว
2. ตรวจสอบว่าวันที่และเวลาไม่ซ้อนทับกับการจองอื่น
3. เวลาทำการ: 10:00 - 20:00 น.
